About the project

The blog started off as a mock up layout made with bootstrap. Then I saw the possiblity to add dynamic posts using Vue.js and therefore decided to start using it in order to separate posts by category. The blog was not created using the Vue.js cli but instead it was integrated with Vue.js through the CDN. The blog uses Vue components for every blog post and works by passing props to each component. The end result is a blog prototype that has posts sorted by categories and is fully reactive.
